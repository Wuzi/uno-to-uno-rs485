# Arduino to Arduino (RS485)

Communication between two arduino using RS485 module

## Transmitter
It's the arduino that will send data to the receiver

## Receiver
It's the arduino that will receive data from the transmitter

## Pins
| RS485 | Arduino |
|-------|---------|
| DI    | 11      |
| RO    | 10      |
| DE/RE | 3       |
